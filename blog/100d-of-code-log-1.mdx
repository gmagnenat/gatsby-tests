**C'est quoi WebGL**

Une API JavaScript  
Rendu de triangles à très hautes vitesse  
Peut être dessiné dans une balise `<canvas>`  
Compatible avec la plupart des browser modernes  
Utilise le gpu de l'ordinateur  
Peut être utilisé pour de la 2D  
Le CPU peut calculer très rapidement mais une seule opération à la fois (1 coeur)  
Le GPU en contrepartie calcule une opération plus lentement mais peut en calculer des milliers en parallèle.  
Un modèle 3d est composé de nombreux triangles, positionnés, avec des couleurs variées. Le GPU positionne ces triangles d'après de nombreux facteurs en une fois.

Les instructions pour placer les points et dessiner les pixels sont écrites dans les **shaders**
