Un objet peut être transformé avec 4 propriétés différentes

- position
- scale
- rotation
- quaternion

<br />

# Position

3 axes, x, y, z pour positionner un object dans l'espace.
Ce n'est pas un objet, c'est un vecteur de la classe **Vector3**

# Optenir la longeur d'un vecteur

    console.log(mesh.position.length());

<br />

# Optenir la distance d'un vecteur à un autre

    console.log(mesh.position.distance(camera.position));

<br />

# Normaliser un vecteur

mettre les valeurs à 1 en gardant la position

    console.log(mesh.position.normalize())

<br />

# Axes Helper

Affichage de trois lignes correspondant aux axes x, y, z

    const axesHelper = new THREE.AxesHelper(2);
    scene.add(axesHelper);
