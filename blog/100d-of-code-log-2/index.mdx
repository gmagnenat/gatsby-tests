# Creation d'une première scene simple

- un fichier index.html
- un fichier script.js
- un script three.min.js

ajouter la librairie Three.js

[https://threejs.org/](https://threejs.org/)

# 3 éléments essentiels

- Des objets
- Une camera
- Un moteur de rendu

Une scene est un container qui comprends des models, des lumières, etc..

# Un objet peut être:

- Primitive geometries
- Imported models
- Particles
- Lights
- Etc..

Pour créer un objet visible on va créer un **Mesh**
C'est une combinaison de **geometry** (la forme) et de **material** (son aspect)

La caméra peut gérer la perspective

# Deux éléments importants:

- field view (largeur de l'angle de vision)
- aspect ratio (en général: largeur du canvas divisé par la hauteur)

Par défaut la caméra est au milieu de la scene.
